## 🤖 Tentang Bot

**VynaaMD** adalah WhatsApp Bot Multi Device yang dibangun di atas library **Baileys** (WhiskeySockets). Bot ini dirancang untuk menjadi asisten serbaguna dengan ratusan fitur mulai dari game, AI, downloader, tools, hingga sistem RPG yang lengkap.

| Info | Detail |
|------|--------|
| 👤 **Author** | VynaaValerie |
| 📱 **Platform** | WhatsApp Multi Device |
| 🔧 **Library** | Baileys (WhiskeySockets) |
| 💻 **Runtime** | Node.js 18+ |
| 🌐 **API** | [api.vtech.biz.id](https://api.vtech.biz.id) |

## 📦 Persyaratan

Sebelum instalasi, pastikan sudah terinstall:

- **Node.js** versi **18.x** atau lebih baru
- **npm** atau **yarn**
- **FFmpeg** (untuk fitur media)
- **ImageMagick** (untuk fitur gambar)
- **Git**

---

## 🚀 Instalasi

### 1. Clone Repository

```bash
git clone https://github.com/VynaaValerie/VynaaMD.git
cd VynaaMD
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Konfigurasi Bot

Edit file `config.js` dan sesuaikan:

```js
global.owner       = ["628xxxx"];      // Nomor owner (tanpa +)
global.nameowner   = "NamaKamu";       // Nama owner
global.numberowner = "628xxxx";        // Nomor owner
global.mail        = "email@kamu.com"; // Email
global.gc          = "https://chat.whatsapp.com/xxx"; // Link grup WA
global.vtech       = "YOUR_apikey";    // API Key dari api.vtech.biz.id
```

> 💡 Dapatkan API Key di [api.vtech.biz.id](https://api.vtech.biz.id)

### 4. Jalankan Bot

```bash
npm start
```

### 5. Scan QR / Pairing Code

Ikuti instruksi di terminal untuk menghubungkan WhatsApp kamu.

---

## ⚙️ Konfigurasi

### Pengaturan Utama (`config.js`)

| Variabel | Default | Keterangan |
|----------|---------|------------|
| `owner` | `["628xxx"]` | Nomor WhatsApp owner |
| `nameowner` | `"VynaaValerie"` | Nama owner |
| `maxwarn` | `"5"` | Maksimal peringatan sebelum kick |
| `autobio` | `false` | Auto update bio WhatsApp |
| `antiporn` | `false` | Auto hapus konten porno |
| `spam` | `false` | Anti spam aktif |
| `gcspam` | `false` | Tutup grup saat spam |
| `vtech` | `"YOUR_apikey"` | API Key VTech |

### Menggunakan Environment Variables (`.env`)

```env
API_KEY_VTECH=isi_api_key_kamu_di_sini
```

Lalu di `config.js`, aktifkan baris:
```js
global.vtech = process.env.API_KEY_VTECH;
```

## 🌐 Cara Deploy

<details>
<summary><b>🐳 Docker</b></summary>

```bash
# Build image
docker build -f deploy/Dockerfile -t vynaamd .

# Jalankan container
docker run -d -p 5000:5000 --name vynaamd vynaamd
```

</details>

<details>
<summary><b>🐙 Docker Compose</b></summary>

```bash
docker-compose -f deploy/docker-compose.yml up -d
```

</details>

<details>
<summary><b>☁️ Heroku</b></summary>

```bash
heroku login
heroku create nama-bot-kamu
git push heroku main
heroku ps:scale web=1
```

</details>

<details>
<summary><b>🚂 Railway</b></summary>

1. Fork repository ini
2. Buka [railway.app](https://railway.app) dan login
3. Buat project baru dari GitHub repo ini
4. Tambahkan environment variable yang diperlukan
5. Deploy!

</details>

<details>
<summary><b>🖥️ Render</b></summary>

1. Fork repository ini
2. Buka [render.com](https://render.com) dan login
3. Buat **Web Service** baru dari GitHub repo ini
4. Set **Build Command**: `npm install`
5. Set **Start Command**: `npm start`
6. Tambahkan environment variables
7. Deploy!

</details>

<details>
<summary><b>🐧 VPS / Linux</b></summary>

```bash
# Install Node.js 18+
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install FFmpeg & ImageMagick
sudo apt install ffmpeg imagemagick webp -y

# Clone & install
git clone https://github.com/VynaaValerie/VynaaMD.git
cd VynaaMD
npm install

# Jalankan dengan PM2 (recommended)
npm install -g pm2
pm2 start index.js --name VynaaMD
pm2 save
pm2 startup
```

</details>
